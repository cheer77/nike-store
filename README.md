# 🛍️ Nike Store (Vanilla JS)

A simple e-commerce project inspired by Nike Store, built with pure JavaScript.

## 🚀 Features

- Product catalog rendering
- Add to cart functionality
- Cart drawer (open/close)
- Increment / Decrement products
- Remove items from cart
- Stock control (out of stock handling)
- Dynamic UI updates (cart, products, header)

## 🧠 Architecture

The project is built using a component-based approach:

- **Cart** – manages application state (products in cart)
- **Products** – renders product list and handles user actions
- **Header** – displays cart count
- **CartDrawer** – renders full cart UI and interactions
- **Data layer** – static product data (later will be replaced with API)

All components are connected through a shared state (Cart).

## 🛠️ Tech Stack

- HTML5
- SCSS
- Vanilla JavaScript (ES6+)
- Vite

## 📦 Future Improvements

- Quantity-based cart logic (better UX)
- API integration (replace static data)
- Animations and micro-interactions
- Persistent cart (LocalStorage)
- Checkout flow

## 💡 Goal

This project is built for learning purposes to understand:

- State management without frameworks
- Component architecture in Vanilla JS
- DOM manipulation and event delegation
